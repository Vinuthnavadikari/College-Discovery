# 🎓 College Discovery Platform

A full-stack web application that helps students discover, compare, and analyze engineering colleges based on fees, ratings, placements, and location.

## 🚀 Live Demo

Deployed on Vercel

https://college-discovery-emsd.vercel.app

## 📂 GitHub Repository

https://github.com/Vinuthnavadikari/College-Discovery

---

## ✨ Features

* Search colleges by name
* Filter colleges by state
* Filter colleges by rating
* Filter colleges by fees
* View detailed college information
* Compare colleges side-by-side
* Responsive user interface
* Cloud-hosted PostgreSQL database

---

## 🛠 Tech Stack

### Frontend

* Next.js 16
* TypeScript
* Tailwind CSS

### Backend

* Next.js API Routes
* Prisma ORM

### Database

* PostgreSQL (Neon)

### Deployment

* Vercel

---

## 📊 Database Fields

Each college contains:

* Name
* Location
* State
* Fees
* Rating
* Average Package
* Highest Package
* Placement Rate
* Description

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Vinuthnavadikari/College-Discovery.git
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
DATABASE_URL=your_database_url
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
